## 🛠 開発ガイド

### 新しいアプリを追加する場合：

1. **ディレクトリ作成**
```bash
mkdir apps/your-new-app
```

2. **基本ファイル作成**
```
apps/your-new-app/
├── index.html          # メインHTML
├── script.js           # JavaScript（オプション）
├── style.css           # CSS（オプション）
└── README.md           # アプリ説明（オプション）
```

3. **トップページに追加**
- `index.html`にアプリカードを追加
- このREADMEファイルにアプリ説明を追加
